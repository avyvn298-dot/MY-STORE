About the project

This template is built for maintainability. It uses semantic HTML, ARIA where necessary, and progressive enhancement: everything works without JavaScript, and JS only improves the experience.

Design choices

  • Single stylesheet to keep the critical path short
  • Responsive grid layout with CSS gap and clamp() for fluid type
  • Accessible interactions and focus styles

Performance tips

  • Pre-compress assets (gzip or Brotli on server)
  • Use modern image formats (AVIF/WebP) and srcset
  • Defer non-essential JavaScript
